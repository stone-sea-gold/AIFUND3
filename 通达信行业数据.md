# 通达信本地行业概念板块数据获取与使用说明

## 一、数据获取方式：自动更新，无需特殊下载

通达信的行业、概念、地域、风格等板块分类数据，**不需要任何特殊手动下载**。只要软件连接过行情主站并进行了每日盘后数据刷新，这些板块文件就会自动更新到最新状态。

- **自动更新时机**：登录交易或行情主站时，通达信会自动同步板块分类信息。  
- **手动补充更新**：如需强制刷新，可进入“系统” → “数据维护” → “更新板块信息”进行即时更新。  

因此，只要保持正常的每日盘后数据下载习惯，本地硬盘上的板块数据就是完整且最新的。

---

## 二、数据存储路径与文件说明

所有板块数据均存放在通达信安装目录下的 `T0002\blocknew\` 文件夹内（部分版本也可能在 `T0002\hq_cache\` 下存在类似文件）。文件格式为自定义二进制 `.dat`，核心文件如下：

| 文件名 | 对应板块类型 | 说明 |
|--------|--------------|------|
| `block_gn.dat` | **概念板块** | 全部概念板块及其成分股 |
| `block_fg.dat` | **风格板块** | 如高价股、破净股、低市盈率等 |
| `block_hy.dat` | **行业板块** | 通达信细分行业分类（部分版本使用） |
| `block_zs.dat` | **指数板块** | 各类指数成分股 |

> 这些文件为纯本地存储，读取完全免费，无调用次数或频率限制。

---

## 三、数据结构：板块→个股的独立模块

通达信板块文件按 **“板块 → 成分股列表”** 的方式组织，是一个独立的分类映射模块，**并非个股行情的附属标签**。

以 `block_gn.dat`（概念板块）为例，其内部二进制结构大致为：

- **文件头**：4 字节（可跳过）
- **循环读取板块记录，直到文件末尾**：
  - 2 字节：板块代码（整数）
  - 9 字节：板块名称（GBK编码，不足9字节补 `\0`）
  - 2 字节：成分股数量 `N`
  - `N × 7` 字节：每只股票 7 字节，前 6 字节为股票代码（如 `600519`），第 7 字节为市场标识

**重要特性**：
- 个股日K线等行情文件（如 `sh600519.day`）中**不包含任何板块标签**。
- 必须通过解析上述板块文件才能知道某只股票属于哪些板块（可通过反向索引一次性构建）。

## 四、数据调取方法

### 方法一：使用开源库 `pyTDX`（推荐）

```python
from pytdx.reader import BlockReader

# 读取概念板块
df = BlockReader().get_df(r"D:\通达信\T0002\blocknew\block_gn.dat")
# df 包含 'blockname'（板块名称）和 'code'（成分股代码）两列
print(df.head())
```

- 读取其他板块只需替换对应的文件名。
- 纯读本地文件，**无频率限制**，但通达信运行中可能锁定文件，建议先复制或关闭软件再读。

### 方法二：自行按格式解析（Python示例）

```python
import struct

def parse_block_dat(filepath):
    blocks = []
    with open(filepath, 'rb') as f:
        f.read(4)  # 跳过文件头
        while True:
            header = f.read(13)
            if not header:
                break
            code_int, name_bytes, stock_num = struct.unpack('<H9sH', header)
            name = name_bytes.decode('gbk').rstrip('\x00')
            codes = []
            for _ in range(stock_num):
                raw = f.read(7)
                codes.append(raw[:6].decode())  # 取6位代码
            blocks.append({'block_name': name, 'codes': codes})
    return blocks

blocks = parse_block_dat('block_gn.dat')
```

## 五、从“板块→个股”到“个股→板块”的转换

原始数据是板块到个股的映射，但通过简单反转即可获得个股的完整板块标签：

```python
# 假设已用 BlockReader 得到 df
stock_to_blocks = {}
for block_name, group in df.groupby('blockname'):
    for code in group['code']:
        stock_to_blocks.setdefault(code, []).append(block_name)

# 查询示例：000001（平安银行）所属所有概念
print(stock_to_blocks.get('000001', []))
```

构建一次后即可全局使用，实现“个股映射到行业概念”的效果。

## 六、使用注意事项

1. **文件编码**：板块名称等中文使用 **GBK** 编码，解析时需指定。
2. **文件锁定**：通达信运行时可能对 `.dat` 文件加锁，直接读取会引发错误。**建议在通达信关闭后读取，或先将文件复制到临时目录再解析**。
3. **更新频率**：板块数据通常每日盘后自动更新一次，成分股变动不会盘中实时同步。
4. **数据覆盖范围**：覆盖全部A股，包括沪深北交所所有正常上市股票。
5. **版权声明**：数据来自通达信免费客户端，仅供个人学习与研究使用，**商业用途需注意通达信的版权许可**。
