# 无情操盘手交易系统

> **定位：** 以白线(DEMA10) + 黄线(quad-MA) 为核心的均线量价交易系统。

## 与波浪系统的关系

| 维度 | 波浪理论系统 | 无情操盘手系统 |
|------|-------------|---------------|
| **核心指标** | MACD(5,34,5) + Fib 回撤 | 白黄线(DEMA10/quad-MA) + KDJ |
| **白黄线** | 相同定义，用于辅助过滤 | 相同定义，用于核心多空判断 |
| **入场逻辑** | Fib 区 + MACD 金叉 + 放量 | B1 超卖 + B2 确认 / B砖反转 |
| **止损逻辑** | 固定价位（波浪结构破位） | 买入当日低点 / 暴力K低点 |
| **持仓风格** | 趋势跟踪，拿到波浪目标 | 严格纪律，3天不涨就走 |
| **黄线以下** | 可操作（超跌反转） | 标准战法禁止 / 碗大战法可做 |
| **死叉处理** | 持仓期清仓（白黄线死叉） | 无条件清仓（最高优先级） |

### 共同铁律
1. 白黄线死叉 = 无条件清仓
2. 白线 = DEMA(10)，黄线 = (MA14+MA28+MA57+MA114)/4
3. 触及止损必须执行，不犹豫不补仓

> **当前持仓操作策略不变**，仍以 [[回测策略操作手册]] 为准。本系统作为辅助参考和学习知识库。
