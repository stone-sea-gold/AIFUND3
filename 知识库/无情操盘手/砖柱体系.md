# 砖柱体系（知行合一 MACD 砖柱系统）

> 来自 [[无情操盘手]] 系统。基于白线和黄线差值构建的 MACD 柱体系统，用四种颜色标识多空能量状态。

---

## 砖的计算

| 指标 | 公式 | 含义 |
|------|------|------|
| DIF | 白线(DEMA10) - 黄线(quad-MA) | 短期趋势与多空线的差值 |
| DEA | EMA(DIF, 9) | DIF 的 9 周期平滑 |
| 砖 | (DIF - DEA) × 2 | MACD 柱状体（能量柱） |

> **注意：** 此处 DIF 的定义与标准 MACD 不同。标准 MACD 的 DIF = EMA(12) - EMA(26)，而砖柱系统的 DIF = DEMA10 - quad-MA。

---

## 砖的四色分类

| 颜色 | 条件 | 含义 | 趋势信号 |
|------|------|------|---------|
| 🔴 红砖 | 砖 > 0 且 砖 > 前一根砖 | 多头能量释放 | 强势上涨 |
| 🟡 黄砖 | 砖 > 0 且 砖 ≤ 前一根砖 | 多头能量衰减 | 涨势放缓 |
| 🟢 绿砖 | 砖 < 0 且 砖 < 前一根砖 | 空头能量释放 | 加速下跌 |
| 🔵 蓝砖 | 砖 < 0 且 砖 ≥ 前一根砖 | 空头能量衰减 | 跌势放缓 |

### 自然转换顺序

```
上涨周期: 蓝砖 → 红砖 → 黄砖（空头耗尽 → 多头释放 → 多头衰减）
下跌周期: 黄砖 → 绿砖 → 蓝砖（多头衰减 → 空头释放 → 空头耗尽）
```

## B砖战法（连续绿砖 → 黄砖 + B2） {#B砖战法}

### 买入条件（全部满足，只做这一种情况）

1. **连续 3~4 根绿砖**（空头能量充分释放，多了不考虑）
2. **绿砖经蓝砖/红砖过渡后转为黄砖**（自然转换顺序：绿→蓝→红→黄）
3. **过渡期间出现 B2 中长阳**（实体 ≥ 3%）确认反转
4. **买入点 = 第一根黄砖日收盘价**

### 关键特性

- **不依赖 B1：** 绿砖→黄砖转换本身就是空头耗尽信号，不需要 J < 15 超卖确认
- **不限均线位置：** B2 可在黄线下方、黄白线间、白线上方均有效
- **止损位：** B2 当日最低点
- **死叉铁律仍为最高优先级**

### 核心逻辑

```
连续绿砖 = 空头能量持续释放，跌势凶猛
    ↓
绿砖转蓝砖 = 空头能量开始衰减
    ↓
蓝砖转红砖 = 多头开始接管
    ↓
红砖转黄砖 = 多头释放后衰减，趋势确认
    ↓
黄砖日 + B2 中长阳 = 底部反转确认 → 买入
```

## 砖柱读盘要点

| 状态 | 操作含义 |
|------|---------|
| 连续红砖 | 多头释放中，持有不动 |
| 红砖转黄砖 | 涨势放缓，注意减仓信号 |
| 连续绿砖 | 空头释放中，空仓等待 |
| 绿砖转蓝砖 | 跌势放缓，开始关注底部信号 |
| 蓝砖→红砖→黄砖 + B2 | B砖买入信号（唯一做多条件） |
