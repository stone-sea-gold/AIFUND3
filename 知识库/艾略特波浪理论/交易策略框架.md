# 交易策略框架

> 来自 [[艾略特波浪理论]] 系统。

---

## 三大送钱行情（调整必剧烈）

```
信号1: 终结楔形 → 破趋势线后急跌
信号2: 衰竭五浪 → 双顶/双底形成后反转
信号3: 五浪延长 → 内部五浪走完后急速调整至五浪2

牛市下跌: 不一定需要放量
熊市上涨: 必须放量确认（否则为假信号）
```

---

## 最赚钱的三个位置

```
位置1: 三浪（主升浪）
位置2: C浪（主跌浪/调整驱动段）
位置3: 三大送钱行情后的反转段
```

---

## 三浪交易法

```
建仓1（试仓）: 二浪底部，走出三浪结构 → 10~20%仓位
  止损: 一浪底部

建仓2（加仓）: 三浪1未破前高回踩不破前低 → 加仓

建仓3（确认加仓）: 三浪1破浪1顶部 → 提至30%仓位
```

---

## 五浪交易法（在四浪中布局）

```
入场: 四浪运行过程中布多单
判断依据:
  - 四浪底 vs 二浪顶 距离大 → 必有新高
  - MACD三浪顶有高点、指标也有高点（配合完美）→ 新高确认
出场: 五浪3出单（不等五浪5，因四浪变化微小易误判）
```

---

## C浪交易法

```
前提: A浪为五浪结构 → B浪反弹浅 → 放弃B浪交易
入场: B浪高点处（B浪ABC走完，A=C时）
止损: A浪起点上方
加仓: 破A浪终点（下方低点）后加仓
```

---

## 反转交易法（123法则）

```
步骤1（信号）: 突破中期趋势线 → 10%仓位试仓
步骤2（确认）: 破四浪低点 → 加仓至30%

近距判断（信号出现后立即交易 vs 等回抽）:
  - 最近高点离趋势线近 → 破位立即进场
  - 最近高点离趋势线远 → 等回抽后进场
```

---

## 资金管理原则

```
试仓: 10%（即使错误不伤元气）
确认: 30%
最大: 30%（绝不满仓追加）
止损: 必设，位置明确
```
