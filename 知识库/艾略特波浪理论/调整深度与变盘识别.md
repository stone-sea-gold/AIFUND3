# 调整深度定位与变盘识别

> 来自 [[艾略特波浪理论]] 系统。

---

## 一、浪4调整支撑位

```
当本身是大四浪调整时:
  支撑位 = 前一浪级的浪4区域（即浪3内部的浪4底部）
  止损位 = 浪1的顶部（破则二四重合，驱动不成立）
```

---

## 二、五浪延长后ABC调整支撑位

```
支撑位 = 五浪2的位置
  → 可能是整个ABC结束（锯齿形）
  → 也可能仅是A浪结束（平台形后续还有BC）
```

---

## 三、价格通道修正

```
标准通道: 最低点 + 次低点连线 → 中间最高点做平行线
修正通道: 当浪4未触及趋势线时
  → 以最后一次接触趋势线的点为新最低点
  → 重新做趋势线 + 平行线
  → 五浪目标位 = 修正后通道上轨
```

---

## 四、变盘识别

```
触发条件: C浪达到A浪的1.618倍且未止跌
位置: 发生在C浪（非三浪即C浪）
验证: 
  - 一浪顶 vs 四浪底 距离大 → 驱动而非调整
  - MACD无背离（价格涨指标也涨）→ 驱动确认
结论: 
  - 平台形调整不会变盘
  - 锯齿形调整C浪可能变盘为新驱动的三浪
```
