# 艾略特波浪理论

## 核心框架速览

```
完整循环 = 驱动5浪(1-2-3-4-5) + 调整3浪(A-B-C)

三条铁律（违反即数浪错误）:
  1. 浪2 < 浪1的100%
  2. 浪3不是最短的
  3. 浪4不进入浪1区域

MACD口诀:
  三浪 → 价格新高，指标新高
  四浪 → 指标回零轴
  五浪 → 价格新高，指标背离
```

## MACD参数

```
MACD(5, 34, 5) — 快线5，慢线34，信号5
```

> **注意：** 非标准参数(12,26,9)。本系统所有计算均使用此参数。

- 长高电新/富维股份/湖北宜化/金风科技 → Wave+MACD 策略（Fib区+MACD金叉）
- 华银电力/汉缆股份 → 趋势跟踪策略（金叉持有，死叉退出）
