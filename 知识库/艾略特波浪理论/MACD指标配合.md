# MACD指标配合体系

> 来自 [[艾略特波浪理论]] 系统。MACD 是波浪分析的核心辅助指标。

---

## 推荐参数

```
MACD参数: 5, 34, 5（快线5，慢线34，信号5）
```

> **注意：** 非标准参数(12,26,9)。本系统所有 MACD 计算均使用 (5,34,5)。

---

## 核心口诀

```
三浪 → 价格创新高，指标创新高
四浪 → 指标回零轴
五浪 → 价格创新高，指标背离
```

---

## 背离规则

```
第一次背离: 出现在三浪顶 vs 五浪顶之间
五浪内部: 最多连续背离3次（一鼓作气，再而衰，三而竭）
  → 五浪延长 或 终结楔形 导致内部有3个高点
```

### 重要原则
- 是**价格见顶引起指标背离**，不是指标背离导致价格见顶
- 指标不背离，价格不见顶
- 背离仅作为信号，不可直接据此开仓交易
- 指标走低时成交量必须缩量
