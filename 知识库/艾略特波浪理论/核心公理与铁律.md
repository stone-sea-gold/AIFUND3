# 核心公理与驱动浪铁律

> 来自 [[艾略特波浪理论]] 系统。

---

## 一、核心公理

| 编号 | 公理 | 说明 |
|------|------|------|
| A1 | 完整循环 = 驱动5浪 + 调整3浪 | 1-2-3-4-5 + A-B-C，共8浪一个周期 |
| A2 | 驱动浪朝大一级趋势同向运动 | 1/3/5浪为推动，2/4浪为回调 |
| A3 | 调整浪永远不是五浪结构 | 若调整出现五浪，则只是调整的一部分（A浪） |
| A4 | 市场循环相似但不相同 | 历史重复的是规律，不是走势本身 |

---

## 二、驱动浪铁律（违反即数浪错误）

### 三条不可违反规则

```
规则1: 浪2回撤 < 浪1的100%
        → 若破浪1起点，则不是调整而是新驱动

规则2: 浪3永远不是1/3/5中最短的
        → 大多数情况下浪3最长（延长浪除外，但仍不能最短）

规则3: 浪4不进入浪1价格区域（一四不重合）
        → 全额市场（股票）：严格不可重合
        → 杠杆市场（期货/外汇）：允许微幅重合
```

### 驱动浪三种形态

#### (1) 楔形（倾斜三角形）

| 属性 | 终结楔形 | 引导楔形 |
|------|----------|----------|
| 出现位置 | 5浪、C浪（行情末端） | 1浪（行情起步） |
| 内部结构 | 3-3-3-3-3（五个三） | 5-3-5-3-5 |
| 二四重合 | 是 | 是 |
| 出现频率 | 常见 | 少见 |
| 共同特征 | 都是五浪结构；都二四重合；都沿同一方向运动 |

**终结楔形确认要点：**
- 五浪常以"翻越"（微创新高/新低）方式结束
- 成交量递减
- MACD背离（三浪顶→指标高点，五浪顶→指标背离）

**引导楔形特征：**
- 筑底过程缓慢
- 完成后二浪回撤可能很深（接近吃掉整个引导楔形涨幅）
- 破下方上升趋势线后→调整剧烈→送钱行情

#### (2) 延长浪

```
规则: 1/3/5浪中有且仅有一个延长浪
频率: 浪3延长 > 浪5延长 > 浪1延长

延长浪内部 = 新的五浪结构，遵循所有驱动浪规则
```

**区分三浪2回调 vs 大四浪调整：**

```
方法一（未破浪1顶时）: 
  比较调整时间
  → 该段调整时间 > 二浪调整时间 → 大四浪
  → 该段调整时间 < 二浪调整时间 → 三浪2（子浪级别更小）

方法二（破浪1顶时）:
  破二浪顶 + MACD指标无背离（价格涨指标也涨）
  → 铁定是三浪2，后续有三浪3拉升
```

**九浪延长（无界定）：** 连续9浪无法分清子浪/母浪，分析方法同五浪延长

#### (3) 衰竭五浪（失败五浪）

```
定义: 五浪未能超过三浪终点
触发条件: 
  - 通常在超强势二浪之后出现
  - 一般需重大数据/事件刺激
结果: 形成双顶/双底 → 中长线反转信号
```

**真正双顶的四个条件（全部满足）：**
1. 最后一次上扬成交量相对前高缩量
2. 两个高点间距 ≥ 1个月
3. 两高点间最低点对股价调整 ≈ 20%
4. 两高点间形成半圆弧/半椭圆形态
