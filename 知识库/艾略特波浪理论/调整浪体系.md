# 调整浪体系

> 来自 [[艾略特波浪理论]] 系统。

---

## 核心原则

```
调整浪永远是三浪结构（非五浪）
若反趋势方向出现五浪 → 仅为调整的一部分（A浪），不是调整结束
调整浪的 A浪 只有两种结构：五浪结构 或 三浪结构
  → A浪五浪 = 锯齿形(535)
  → A浪三浪 = 平台形(335)
B浪永远是三浪结构
C浪永远是五浪结构
```

---

## 四大调整类型

### (1) 锯齿形调整（535）

```
结构: A(5浪) - B(3浪) - C(5浪)
A浪: 内部五浪，二四不重合
B浪: 反弹浅，常在A浪的38.2%/50%/61.8%结束
      B浪反弹不可能到达A浪起点（否则数浪错误）
C浪: 五浪结构驱动下跌
比例: A = C（AC等长）
```

**联合锯齿形调整：**
- 时间短+幅度浅 → 调整未完成 → X浪连接 → 第二组锯齿形
- 两组锯齿形合计需满足时间空间对称

### (2) 平台形调整（335）— 三种变体

| 类型 | A浪 | B浪 | C浪 | 比例关系 |
|------|------|------|------|----------|
| 规则平台 | 3浪 | 反弹至A起点附近 | 跌至A终点附近 | A = B = C |
| 扩张平台（强势调整） | 3浪 | 超过A起点 | 远超A终点 | A × 1.618 = C |
| 顺势平台 | 3浪 | 超过A起点（最长浪） | 未超A终点 | A = C，B最长 |

```
扩张平台 B浪高点测算: B浪内部 A = C（B浪子浪的AC等长）
顺势平台: 常出现在四浪，前提是三浪拉升极快
```

### (3) 三角形调整

**收缩三角形（常见）：**

| 类型 | 方向偏好 | 水平线位置 |
|------|----------|------------|
| 对称三角形 | 上/下各50% | 无水平线 |
| 上升三角形 | 向上突破70-80% | 上边水平 |
| 下降三角形 | 向下突破70-80% | 下边水平 |

**三角形分析五要素：**

```
1. 内部结构: ABCDE五浪，每浪均为三浪结构
2. 成交量: 临近突破点递减 → 突破瞬间放量（向上突破必须放量）
3. 突破区间: 入场点到原点的 1/2 ~ 3/4 区间内突破最有效
4. 目标位（衡量定则）: 底部长度 从突破点往上加
5. 失败信号:
   - 价格穿过原点（压缩挤过）→ 失败
   - 内部成交量不规则 → 假信号
   - 突破后未超前高/前低又回落 → 假信号
```

**涨前抛售：** 下降三角形向下突破时放大量 → 不是真跌，是抛售后即将上涨

**顺势三角形：** B浪超过A浪起点，内部仍为五浪ABCDE结构

### (4) 联合型调整（双重浪/三重浪）

```
特征:
  - 横盘震荡为主
  - 最常出现在四浪
  - 两个浪形之间用X浪（三浪结构）连接
  - 前后浪形不同类型（交替原则）
  - 二浪极少出现
```
