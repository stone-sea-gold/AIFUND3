# 交替原则与比例关系

> 来自 [[艾略特波浪理论]] 系统。

---

## 一、交替原则

### 驱动浪交替（浪2 vs 浪4）

| 维度 | 浪2 | 浪4 |
|------|------|------|
| 深度 | 深（61.8%） | 浅（38.2%） |
| 形态 | 锯齿形居多 | 平台/三角形居多 |
| 复杂度 | 简单 | 复杂（数不清常是四浪） |
| 时间 | 短 | 长 |

**延长浪交替：** 1/3/5中有且仅有一个延长，三浪延长概率最大

### 调整浪交替（A浪 vs B浪 内部形态）

```
若A浪内部 = 平台形 → B浪内部 = 锯齿形
若A浪内部 = 锯齿形 → B浪内部 = 平台形
ABC不可能全简单或全复杂，复杂度递进交替
```

## 二、比例关系体系

### 回撤（调整浪深度）

```
浪2 回撤 浪1: 常见 50% ~ 61.8%（剧烈调整）
浪4 回撤 浪3: 常见 38.2%（盘整型）
B浪 回撤 A浪:
  - 锯齿形: 38.2% ~ 50%（B浪反弹浅）
  - 平台形: 接近100%（B浪反弹至A起点附近）
X浪: 约 38.2%
```

### 倍数（驱动浪目标位）

```
浪3目标: 浪1 × 1.618（从浪2底往上加）
浪5目标（三浪延长时）: 浪5 = 浪1（等长），从浪4底往上加
浪5目标（五浪延长时）: (浪1底→浪3顶) × 1.618，从浪4底往上加
浪5目标（一浪延长时）: 浪1 × 0.618 = 浪2底→浪5
```

### 调整浪比例

```
锯齿形: A = C（AC等长），偶见 C = A × 0.618
平台形（规则）: A = B = C
平台形（扩张）: A × 1.618 = C
三角形: C = A × 0.618, E = D × 0.618
```

### 时间与空间对称

```
调整时间参照:
  - 调整时间 ≈ 驱动时间的 1/2 或 1:1

空间换时间: 调整深（61.8%）→ 时间短
时间换空间: 调整浅（38.2%）→ 时间长
时间短 + 空间浅 → 调整未完成 → 联合型调整将出现
```
