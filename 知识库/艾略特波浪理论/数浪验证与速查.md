# 数浪验证清单与波浪个性速查

> 来自 [[艾略特波浪理论]] 系统。

---

## 数浪验证清单

```
□ 浪3是否为1/3/5中最长（至少不是最短）？
□ 浪2是否未破浪1起点？
□ 浪4是否未进入浪1区域（考虑市场类型）？
□ 浪3是否超过浪1终点？
□ 调整浪是否为三浪结构（非五浪）？
□ A浪结构（三浪/五浪）是否与后续BC形态匹配？
□ 交替原则是否满足（深浅、形态、复杂度）？
□ MACD是否配合（三浪高→四浪零轴→五浪背离）？
□ 成交量是否配合（驱动放量、调整缩量）？
□ 时间与空间是否对称？
□ 外表形态与内部结构是否一致？
```

## 波浪个性速查

| 浪 | 个性特征 | 成交量 | 交易建议 |
|----|----------|--------|----------|
| 1浪 | 打底过程，50%被忽略，常被误认为熊市反弹 | 低 | 难交易，引导楔形出现则底部扎实 |
| 2浪 | 回撤深（几乎吃掉1浪涨幅），锯齿形居多 | 低量结束 | 不交易，等待确认 |
| 3浪 | 最强劲，信心爆棚，跳空/突破频现 | 大幅放量 | 主力交易浪，从三浪2开始建仓 |
| 4浪 | 横盘为主，数不清，为5浪筑底也为行情筑顶 | 缩量 | 布局五浪多单的时机 |
| 5浪 | 强弩之末，速度<3浪，最后的末班车 | 量能衰减 | 五浪3出场，不贪 |
| A浪 | 为BC定调子，五浪→锯齿形，三浪→平台形 | — | 不主动交易，观察结构 |
| B浪 | 假牛市/陷阱，成交量萎缩 | 逐渐萎缩 | 不交易（锯齿形B浪放弃） |
| C浪 | 摧毁性极强，具备三浪特征 | 放量 | 最赚钱的浪之一，重点交易 |
